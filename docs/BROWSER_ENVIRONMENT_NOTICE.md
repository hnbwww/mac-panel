## 🎯 新增功能

### 使用环境条件提示栏

在浏览器页面底部添加了一个信息提示栏，显示当前的使用环境条件和限制说明。

### 功能说明

**位置**：浏览器页面底部（Card 下方）

**样式**：蓝色信息提示框（Alert 组件）

**特性**：
- 可关闭（点击关闭按钮）
- 显示图标
- 清晰的层级说明
- 12px 小字号，不占用太多空间

### 提示内容

```
💡 使用环境条件 & 限制说明

• 运行在非 HTTPS 环境（http://192.168.0.7），剪贴板 API 受限 → 使用手动输入方式粘贴
• 浏览器画面是截图（PNG图片），每次刷新约 0.5 秒，存在输入延迟
• 选中文字后自动复制到剪贴板历史，可在"剪贴板"面板查看
• 网页复制的内容可点击"复制到本机"按钮，复制到本地剪贴板使用
• 所有键盘输入通过 WebSocket 发送到远程浏览器，请确保网络连接稳定
```

### 提示详情

#### 1. 非 HTTPS 环境说明

**内容**：
```
• 运行在非 HTTPS 环境（http://192.168.0.7），剪贴板 API 受限 → 使用手动输入方式粘贴
```

**原因**：
- 浏览器的 Clipboard API 只在 HTTPS 或 localhost 环境下可用
- 当前运行在 `http://192.168.0.7`（非 HTTPS）
- 无法直接访问本地剪贴板

**解决方案**：
- 使用弹出的输入框手动输入要粘贴的文本
- 后端会将输入的文本粘贴到远程浏览器

#### 2. 截图延迟说明

**内容**：
```
• 浏览器画面是截图（PNG图片），每次刷新约 0.5 秒，存在输入延迟
```

**原因**：
- 浏览器画面是截图（静态图片）
- 每 0.5 秒刷新一次（2 FPS）
- 输入操作需要等待下一次截图才能看到结果

**用户体验**：
- 输入文字后需要等待约 0.5 秒
- 滚动页面也需要等待刷新
- 这是远程浏览器的正常延迟

#### 3. 剪贴板历史说明

**内容**：
```
• 选中文字后自动复制到剪贴板历史，可在"剪贴板"面板查看
```

**功能**：
- 拖拽选择文字后自动复制
- 自动保存到剪贴板历史
- 点击工具栏"剪贴板"按钮查看
- 最多保存 20 条历史记录

#### 4. 复制到本机说明

**内容**：
```
• 网页复制的内容可点击"复制到本机"按钮，复制到本地剪贴板使用
```

**功能**：
- 来自网页的记录有"复制到本机"按钮
- 一键复制到本地剪贴板
- 在本地任何地方粘贴使用
- 方便远程内容本地使用

#### 5. 网络连接说明

**内容**：
```
• 所有键盘输入通过 WebSocket 发送到远程浏览器，请确保网络连接稳定
```

**重要性**：
- 所有输入都通过 WebSocket 发送
- 需要稳定的网络连接
- 网络不稳定会影响输入响应
