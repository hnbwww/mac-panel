# 浏览器剪贴板历史功能

## 更新时间
2026-03-06 17:28

## 🎉 新功能：剪贴板历史管理

### 功能说明

添加了完整的剪贴板历史管理面板，可以查看、管理和使用所有复制的内容。

### 为什么需要剪贴板历史？

**问题**：
1. 选中文字后自动复制，但看不到复制的内容
2. 复制了多次后，只能使用最后一次的内容
3. 无法查看之前复制了什么
4. 想要重复使用之前复制的内容很麻烦

**解决方案**：
- 添加剪贴板历史面板
- 显示所有复制记录
- 支持从历史中选择粘贴
- 支持删除单条记录
- 支持清空所有历史

### 功能特性

#### 1. 自动记录复制内容

**触发时机**：
- 选中文字后自动复制
- 手动粘贴时（通过右键菜单或输入框）

**记录内容**：
- 复制的文本内容
- 复制的时间戳
- 内容预览（前100个字符）

#### 2. 剪贴板按钮

**位置**：浏览器工具栏
**图标**：剪贴板图标
**徽章**：显示历史记录数量

```
[剪贴板 (3)]  // 数字表示有3条历史记录
```

#### 3. 剪贴板面板

**打开方式**：
- 点击工具栏的"剪贴板"按钮

**面板布局**：
```
┌─────────────────────────────────┐
│ 📋 剪贴板历史  [3 条记录] [清空] │
├─────────────────────────────────┤
│ ┌─────────────────────────────┐ │

│ │                    [粘贴][删除]│ │
│ └─────────────────────────────┘ │
└─────────────────────────────────┘
```

#### 4. 操作按钮

每条历史记录有两个操作按钮：
- **粘贴** - 将该内容粘贴到浏览器输入框
- **删除** - 删除这条历史记录

面板顶部有：
- **清空** - 清空所有历史记录（有确认提示）

```

```

### 技术实现

#### 数据结构

```typescript
interface ClipboardItem {
  id: string;        // 唯一ID（时间戳）
  content: string;   // 完整内容
  timestamp: number; // 时间戳
  preview: string;   // 预览（前100字符）
}
```

#### 状态管理

```typescript
const [clipboardVisible, setClipboardVisible] = useState<boolean>(false);      // 面板显示状态
const [clipboardHistory, setClipboardHistory] = useState<ClipboardItem[]>([]);  // 历史记录列表
const [currentClipboard, setCurrentClipboard] = useState<string>('');           // 当前剪贴板内容
```

#### 添加到历史

```typescript
const addToClipboardHistory = (content: string) => {
  const newItem: ClipboardItem = {
    id: Date.now().toString(),
    content: content,
    timestamp: Date.now(),
    preview: content.length > 100 ? content.substring(0, 100) + '...' : content
  };

  // 添加到列表开头，保留最近20条
  setClipboardHistory(prev => [newItem, ...prev].slice(0, 20));
  setCurrentClipboard(content);
};
```

#### 从历史粘贴

```typescript
const pasteFromHistory = (content: string) => {
  if (wsRef.current) {
    wsRef.current.send(JSON.stringify({
      type: 'paste',
      data: { text: content }
    }));
    message.success('已粘贴');
    setClipboardVisible(false);  // 关闭面板
  }
};
```

#### 删除记录

```typescript
const deleteClipboardItem = (id: string) => {
  setClipboardHistory(prev => prev.filter(item => item.id !== id));
  message.success('已删除');
};
```

#### 清空历史

```typescript
const clearClipboardHistory = () => {
  Modal.confirm({
    title: '清空剪贴板历史',
    content: '确定要清空所有剪贴板历史记录吗？',
    onOk: () => {
      setClipboardHistory([]);
      setCurrentClipboard('');
      message.success('已清空');
    }
  });
};
```

<List.Item
  actions={[
    <Button type="primary" size="small" onClick={() => pasteFromHistory(item.content)}>

### 已知限制

1. **会话级别存储**
   - 历史记录只保存在当前会话
   - 刷新页面后会丢失
   - 未来改进：使用 localStorage 持久化

2. **模拟复制内容**
   - 选中文字自动复制时，无法获取真实内容
   - 显示为占位符文字
   - 只能手动粘贴时获取真实内容

3. **数量限制**
   - 最多保留 20 条记录
   - 超过后自动删除最早的
